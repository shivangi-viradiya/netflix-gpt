🎬 Netflix-GPT

A Netflix-inspired movie browsing application built with React, Redux, Tailwind CSS, Firebase Authentication, and TMDB API — featuring a custom GPT-style Movie Search powered by intelligent mock logic + TMDB integration.

🚀 Live Features

🔐 Authentication

- Firebase Authentication (Sign In / Sign Out)
- Protected routes
- Auth state management with Redux

🎥 Browse Page

- Hero section with featured movie
- Now Playing, Popular, Top Rated, Upcoming rows
- Reusable MovieList + MovieCards components
- Smooth horizontal scroll (Netflix-style)

🤖 GPT Search Page

- Intelligent movie search experience
- Mock GPT logic to generate movie suggestions
- Fetches real movie data from TMDB based on generated names
- Reuses MovieList component to render results
- Dynamic rows per suggestion

🌍 Multi-language Support (Bonus)

- English
- Hindi
- Spanish
- Dynamic placeholder & button text
- Language managed via Redux (configSlice)
